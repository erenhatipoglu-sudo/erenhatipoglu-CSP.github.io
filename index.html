4-Bit Binary Converter

This SwiftUI project displays a 4-bit binary representation of numbers from 0 to 15. When the value exceeds 15, it resets back to 0. Each bit is visually represented by colored rectangles — green for 1 and gray for 0.

Show Swift code 
import SwiftUI

struct ContentView: View {
    @State var id = 0
    let maxID = 15
    
    var body: some View {
        VStack(spacing: 20) {
            Text("Decimal: \(id)")
            HStack(spacing: 10) {
                ForEach(0..<4) { i in
                    let bit = (id >> (3 - i)) & 1
                    Rectangle()
                        .fill(bit == 1 ? Color.green : Color.gray)
                        .frame(width: 40, height: 60)
                }
            }
            HStack(spacing: 20) {
                Button("Enroll (+1)") {
                    id = (id == maxID) ? 0 : (id + 1)
                }
                Button("Reset (0)") {
                    id = 0
                }
            }
        }
        .padding()
    }
}

Scratch Tic-Tac-Toe

A classic Tic-Tac-Toe game made in Scratch. Two players play on the same screen; the game checks rows, columns and diagonals and announces the winner.

Play on Scratch
Documentation 
Purpose: Simple two-player Tic-Tac-Toe to learn event handling and game logic.
How it works:
- 9 tiles toggle between X and O when clicked.
- The game checks rows, columns and diagonals for a win.
- When a win or draw occurs, input stops and the result is shown.

Next steps:
- Add win/draw counters and reset button.
- Implement player vs CPU mode and sounds.
